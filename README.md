🚀 ProTasker – MERN App

🧠About Project - 
A full-stack MERN application for managing tasks and orders with authentication and file upload system.

🛠️ Tech Stack -
React, Node, Express, MongoDB, JWT, Multer

✨ Features - 
User signup/login (JWT auth)
Task CRUD operations
Order management system
Stock update on order
File upload using Multer
Basic backend API structure (MVC)

📁 File Upload - 
Files stored on server (uploads folder)
File path saved in MongoDB  

⚙️ Setup - 
git clone
npm install
npm run dev
